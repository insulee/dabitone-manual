# FNT

> **FNT** — Font 파일. 컨트롤러 내부에 저장되어 **텍스트 렌더링**에 사용되는 **16×16 비트맵 폰트**. 대부분의 경우 내장 기본 폰트로 충분하지만 특수 로고·기호용으로 커스텀 폰트를 업로드할 수 있습니다.

## 왜 16×16 고정?

레거시 LED 컨트롤러 펌웨어 호환성 + 모듈 해상도(한 글자 = 16×16 픽셀)에 맞춘 표준. 한글·한자·기호 등 넓은 범위를 지원하면서도 컨트롤러 메모리 효율이 높습니다.

> [!info] 최근 풀컬러 컨트롤러에서는 **확장 폰트**(32×32 등)를 별도 지원하기도 합니다. 제품별 사양을 확인하세요.

## 어디서 만들어지나

일반적으로 다빛솔루션에서 **제공한 FNT 파일**을 그대로 사용합니다. 커스텀 제작 시:

- 다빛솔루션 전용 폰트 편집 도구 (별도 제공)
- 비트맵 이미지를 FNT로 변환하는 외부 유틸

사용자가 직접 만들 일은 드뭅니다.

## 어디에 저장되나

### 전송 전 (PC)

```
C:\Users\[사용자]\Documents\DabitOne\Fonts\
확장자: .FNT
```

### 전송 후 (컨트롤러)

컨트롤러 내부 폰트 슬롯에 저장. 보통 **기본 한글·영문 폰트**가 공장 출하 시 이미 들어 있음.

## 업로드 방법

[설정 → 폰트 전송](/02-settings/font)에서:

1. FNT 파일 선택
2. 저장할 **슬롯 번호** 선택 (기존 폰트를 덮어쓸지 주의)
3. [전송] 클릭 → 컨트롤러로 업로드

## 사용 방법

폰트를 업로드했다면 텍스트 편집 시 **글꼴 드롭다운**에서 해당 폰트 선택 가능.

## 제약

| 항목 | 값 |
|------|-----|
| 해상도 | 16×16 (기본) 또는 확장 사양 |
| 문자 범위 | 유니코드 또는 KSC 5601 (한글) |
| 슬롯 수 | 컨트롤러별 다름 (보통 1~4개) |
| 파일 크기 | 폰트당 수십 KB~수백 KB |

## 호환성

- 기존 DabitChe FNT 파일 **그대로 호환**
- 컨트롤러 펌웨어가 해당 FNT 포맷을 지원해야 함

## 대안

다빛솔루션에서 제공하는 **기본 한글·영문 폰트**가 대부분의 용도를 커버하므로 커스텀 FNT는 다음과 같은 경우에만 권장:

- 회사 **로고·특수 기호**가 들어간 텍스트 표출
- 비표준 서체 필요
- 디자인 통일성
