# 이미지 편집

> 경로: **[편집] → 이미지 모드**

## 이 화면의 역할

**BMP·PNG·JPG** 이미지를 불러와 전광판 해상도·색상 깊이에 맞게 변환하고 [DAT](/file-formats/dat)로 저장. 로고·아이콘·사진 표출에 사용.

## 기본 사용

1. **[불러오기]** 또는 드래그 앤 드롭으로 이미지 가져오기
2. 크기·위치 조정 (드래그 또는 숫자 입력)
3. (선택) 색상·팔레트 편집
4. (선택) 효과 설정
5. **[저장]** → `.DAT` 생성

## 화면 구성

### 툴바

| 요소 | 유형 | 설명 |
|------|-----|-----|
| **[불러오기]** | Button | 파일 탐색 (또는 드래그 앤 드롭) |
| **[크롭]** | ToggleButton | 이미지 일부만 잘라 사용 |
| **[중앙 정렬]** | Button | 캔버스 중앙으로 이동 |
| **[크기 조정]** | Slider | 배율 조정 (10%~300%) |
| `회전` | ComboBox | 0 / 90 / 180 / 270도 |
| `플립` | ToggleButton | 좌우·상하 뒤집기 |

### 팔레트 편집 (색상 깊이 낮을 때)

1비트·3비트 모드에서 **어떤 색을 켜짐으로 할지** 수동 조정. 예: 빨간 글자만 표출되도록 빨강 채널만 켜기.

### 미리보기

전광판 실제 크기로 렌더된 결과를 실시간 표시.

## 주요 옵션

### 자동 리사이즈

전광판 해상도보다 큰 이미지는 자동으로 축소. 비율 유지 옵션 체크 권장.

### 색상 깊이 변환

RGB 24비트 원본을 3비트·8비트 등으로 **축약**할 때 **디더링**(dithering) 적용 여부 선택 가능:

- **디더링 ON**: 그라데이션이 부드럽지만 패턴이 보일 수 있음
- **디더링 OFF**: 색상 밴드가 뚜렷하지만 깔끔

### 투명 PNG

투명 배경이 있는 PNG를 불러오면 **검정 또는 지정 색**으로 채워짐. 배경색 선택 가능.

### 효과

텍스트와 동일하게 입장·퇴장·속도 효과를 줄 수 있음. 고정 표시하려면 "즉시".

## 저장·내보내기

- `Ctrl+S` — DAT 저장
- 효과 포함 시 자동으로 [ANI](/file-formats/ani)
- 편집 원본은 DabitOne 내부 포맷으로 별도 저장(재편집 가능)
