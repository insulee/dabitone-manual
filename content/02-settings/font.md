# 폰트 전송

> [!abstract] 컨트롤러 폰트 슬롯에 사용자 FNT 파일 업로드
> `설정` 탭 > `폰트` 섹션. 로고·특수 기호용 커스텀 폰트가 필요할 때만 사용.

### 화면 구성

- `파일 선택` — `.FNT` 파일 선택 대화상자
- `슬롯 번호` — 저장할 컨트롤러 폰트 슬롯 (1~4)
- `현재 슬롯 정보` — 해당 슬롯에 들어 있는 폰트 이름
- `전송` — 업로드 시작
- 진행률 바 — 업로드 진행 표시

### 절차

- `설정` 탭 > `폰트` 섹션
- `파일 선택` → 로컬 `.FNT` 파일 지정
- `슬롯 번호` 선택 (보통 2~4, 슬롯 1은 기본 한글 폰트)
- `전송` 클릭 → 진행률 바 확인
- 업로드 완료 후 `편집` 탭 > `텍스트`의 `글꼴` 드롭다운에서 확인

### 슬롯 관리

- `슬롯 1` — 기본 한글 폰트 (덮어쓰기 주의)
- `슬롯 2~4` — 추가 폰트용

> [!warning] 슬롯 1을 덮어쓰기 전 원본 백업 필수. 기본 폰트가 사라지면 기존 저장된 메시지가 깨질 수 있음.

> [!info] FNT 파일 1개당 수십~수백 KB. 컨트롤러 폰트 저장공간이 제한적이므로 큰 폰트 다수 업로드 시 다른 콘텐츠 공간이 줄어든다.
