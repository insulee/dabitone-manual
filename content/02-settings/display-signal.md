# 표출 신호


> 경로: **[설정] → 표출 신호 섹션**

## 이 화면의 역할

모듈의 **신호 송수신 세부 설정**. 화면 크기·색상 깊이만으로 색상이 정확히 안 나올 때 여기서 추가 조정. 대부분 기본값이면 충분하지만 특수 모듈·혼합 구성에서는 필수.

## 기본 사용

기본값으로 먼저 테스트하고, 색상이 이상하면 `Color Order`와 `Scan Order`를 모듈 사양에 맞춰 수정.

## 화면 구성

| 요소 | 유형 | 설명 | 기본값 |
|------|-----|-----|-------|
| `Color Order` | ComboBox | R/G/B 색상 라인 순서 (RGB, RBG, BRG 등) | `RGB` |
| `Scan Order` | ComboBox | 모듈 내부 스캔 방식 (1/4, 1/8, 1/16, 1/32) | `1/16` |
| `Data Rate` | ComboBox | 신호 속도 (Normal / Fast / Slow) | `Normal` |
| `자동 전송 간격` | NumericUpDown | 주기적 자동 재전송 (초) | `0` (사용 안 함) |
| `Gamma` | ComboBox | 감마 보정값 (1.0 ~ 2.8) | `2.2` |

## 주요 옵션

### Color Order

모듈 내부 배선에 따라 색상 순서가 다를 수 있음.

- **RGB**: 표준 (대부분 모듈)
- **RBG, BRG, BGR, GRB, GBR**: 특수 모듈 또는 재배선된 모듈

색상이 바뀌어 나온다면 여기 순서를 바꿔 시도. [표시가 깨질 때](/troubleshooting/02-display-corruption)의 "색상 밀림" 참조.

### Scan Order

LED 모듈이 한 번에 몇 줄을 동시에 스캔하는지. 모듈 사양서에 명시됨.

- **1/4**: 구형 옥외 모듈
- **1/8**: 중급 옥외 모듈
- **1/16**: 표준 (대부분)
- **1/32**: 고해상도 실내 모듈

잘못 선택하면 **가로선이 끊어져** 보이거나 **밝기가 일정하지 않음**.

### 자동 전송 간격

`0`이면 사용자가 명시적으로 전송해야만 화면 갱신. `> 0` 초를 입력하면 그 주기로 DabitOne이 자동 재전송.

> [!warning] 자동 전송 간격을 너무 짧게(예: 1초) 잡으면 네트워크·컨트롤러에 부하. 보통 `30초` 이상 권장.

### Gamma

화면 밝기 곡선. 낮을수록 중간 톤이 밝음. 옥외용은 `2.2~2.5`가 일반적.

## 설정 영속성

컨트롤러 비휘발 메모리에 저장. DabitOne UI는 마지막 값을 기억.
