# Serial

> [!abstract] 시리얼 케이블로 컨트롤러를 직접 연결할 때 사용
> `통신` > `Serial` 그룹박스에서 포트와 속도를 선택한 뒤 `연결 테스트` 클릭.

### 연결 절차

- `통신` > `Serial` 라디오 버튼 선택
- `포트` 드롭다운에서 COM 번호 선택
- `속도` 드롭다운에서 컨트롤러와 동일한 값 선택 (기본 `115200`)
- (RS-485 배선이면) `RS-485 Address` 체크 후 주소값 선택
- `연결 테스트` 버튼 클릭

### 보조 버튼

#### `속도 찾기`
- 현재 포트에서 통신되는 속도를 자동 탐색
- 컨트롤러 속도를 모를 때 사용

#### `장치관리자`
- Windows 장치관리자 창을 바로 엶
- 새 USB-Serial 어댑터 인식 확인용

### 결과 확인

- `연결 테스트 성공` 토스트가 뜨면 시리얼 통신 연결 완료
