# v1.1.0 — 2026-04-21

> DabitChe에서 **DabitOne**로 전환하는 첫 큰 마일스톤 릴리즈입니다. 익숙했던 기능 대부분은 그대로, 사용 경험은 전면 재설계됐습니다.

## Highlights

### 🎯 통신 설정 통합

예전엔 Serial·TCP·UDP·BLE·MQTT가 각자 별도 창·대화상자였습니다. 이제 **한 화면에 모두 모여** 라디오 버튼으로 즉시 전환됩니다. 연결 방식을 여러 번 바꿔 테스트하는 현장에서 체감이 큽니다.

- 관련 체험: [/quickstart/01-connect/](/quickstart/01-connect/)
- 관련 UI: [통신 개요](/docs/01-communication/overview)

### 🎬 내장 GIF 편집기 (신규)

기존에는 외부 툴로 GIF를 만들어 불러와야 했지만, 이제 **타임라인 기반 프레임 편집기**가 내장됐습니다. 프레임 단위 추가·복사·삭제, 프레임별 표시 시간 설정, 효과 전환 모두 앱 안에서 완결.

- 관련 체험: [/quickstart/04-edit-image/](/quickstart/04-edit-image/)

### ✅ 자동 크기 검증

화면 크기 설정 시 입력한 가로·세로 모듈 수가 실제 전광판 물리 구성과 **크게 어긋나면 경고**가 뜨고 안전 기본값을 제안합니다. 예전엔 잘못된 값을 그대로 전송해서 화면이 깨지는 사례가 많았습니다.

### 🎨 디자인 시스템 전면 재설계

WPF 기반 새 UI 스택으로 전환. Windows 10/11에서 **Pretendard 폰트**, 부드러운 그림자·모서리, 다크 모드 지원. 키보드 포커스 표시 개선으로 접근성도 한 단계 업그레이드.

- 관련 UI: [테마](/docs/05-advanced/theme)

### 📁 드래그 앤 드롭

편집기에서 이미지·폰트 파일을 **드래그로 바로** 가져올 수 있습니다. 파일 탐색 → 찾기 → 확인 대화상자의 3단계가 사라졌습니다.

## 그 외 개선

- 전송 진행률·실패 로그 가독성 향상
- 컨트롤러 시계 NTP 자동 동기화 옵션
- 펌웨어 업데이트 진행률·검증 강화
- `AppSettings` 자동 영속성 — 마지막 사용 값 자동 복원
- 단축키 개선 (Ctrl+Enter 전송, Ctrl+S 저장 등)

## 호환성

| 항목 | 상태 |
|------|------|
| 기존 DabitChe `.DAT` 파일 | ✅ 그대로 읽기 가능 |
| 기존 `.ANI` 파일 | ✅ 그대로 읽기 가능 |
| 기존 `.PLA`·`.BGP` 파일 | ✅ 그대로 읽기 가능 |
| 기존 폰트 파일 `.FNT` | ✅ 그대로 사용 가능 |
| 기존 프로토콜(컨트롤러 펌웨어) | ✅ 하위 호환 유지 |

기존 프로젝트 폴더를 그대로 열면 모든 파일이 인식됩니다. 마이그레이션 작업 없음.

## 설치

기존 DabitChe와 **별도 설치**됩니다. 같은 PC에 두 버전을 공존시킬 수 있으니, 익숙해질 때까지 병행 사용 가능.

- 설치: [설치 가이드](/docs/getting-started/install) 참조
