# 테마

> [!abstract] DabitOne 앱 UI 테마 (라이트·다크·시스템 연동)
> `고급` 탭 > `테마` 섹션. 전광판 표출과는 무관, 사용자 편의용.

### 화면 구성

- `테마` — `라이트` / `다크` / `시스템 연동` 라디오
- `액센트 색상` — 강조 색상 (기본 `#2563EB`)
- 미리보기 — 선택 테마 UI 스와치

### 테마 종류

- `라이트` — 밝은 배경, 진한 텍스트
- `다크` — 어두운 배경, 밝은 텍스트 (야간 작업 적합)
- `시스템 연동` — Windows 라이트/다크 모드 자동 반영

선택 즉시 전환. 재시작 불필요.

### 액센트 색상

기본 파랑 외에 팀·브랜드 색상으로 변경 가능. UI 버튼·링크·강조 영역에 반영.

> [!info] 액센트가 너무 밝거나 어두우면 가독성 저하. 미리보기에서 충분히 확인 후 적용.

> [!info] 선택한 테마는 `AppSettings`에 저장. 다음 실행 시 자동 복원.
