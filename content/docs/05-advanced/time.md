# 시간 설정

> [!abstract] 컨트롤러 실시간 시계(RTC)를 PC 시간 또는 NTP로 동기화
> `고급` 탭 > `시간` 섹션. 시간 기반 스케줄(PLA·자동 밝기·전원)이 RTC 기준으로 동작.

### 화면 구성

- `현재 시각` — 컨트롤러 RTC 실시간 표시
- `PC 시각` — DabitOne PC 현재 시간
- `PC 시간으로 설정` — 즉시 PC 시간 덮어쓰기
- `NTP 사용` — 자동 동기화 활성화 체크
- `NTP 서버` — 서버 주소 (기본 `pool.ntp.org`)
- `주기` — 1일 / 1주 / 1개월
- `시간대` — UTC+9 (Asia/Seoul) 등

### PC 시간 동기화

- `현재 시각` 확인
- `PC 시간으로 설정` 클릭 → PC 현재 시간을 컨트롤러에 전송

### NTP 자동 동기화 (권장)

- `NTP 사용` 체크
- `NTP 서버` 입력 (기본 `pool.ntp.org`)
- `주기` 선택
- `저장` → 컨트롤러가 주기적으로 자동 동기화

### NTP vs PC 동기화

- `NTP` — 인터넷 접속 가능 시 자동 지속 동기화. 가장 안정적
- `PC 수동` — 인터넷 없는 현장. 주기적 현장 방문 재동기화 필요

### 시간대

- 한국 — `UTC+9 (Asia/Seoul)`
- DST 없음 (다른 지역 설치 시 해당 국가 DST 확인)

> [!warning] 시계가 어긋나면 자동 밝기·전원·PLA 스케줄이 의도한 시각과 다른 시간에 동작. 정확한 동기화 필수.

> [!info] NTP 미사용 환경에서는 RTC 배터리(보통 수년) 만료 시 시간 리셋. 최소 월 1회 PC 동기화 권장.
