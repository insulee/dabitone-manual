# 펌웨어

> [!abstract] 컨트롤러 펌웨어 버전 조회·`.bin` 파일 기반 업데이트
> `고급` 탭 > `펌웨어` 섹션.

### 화면 구성

#### 정보 섹션
- `모델` — 컨트롤러 모델명 (예: DABIT-PRO-v3)
- `현재 버전` — 펌웨어 버전 (예: 2.4.1)
- `빌드 일자` — 펌웨어 빌드 날짜
- `최신 버전` — (선택) 제조사 서버 최신
- `펌웨어 정보 읽기` — 정보 새로고침

#### 업데이트 섹션
- `파일 선택` — `.bin` 파일 선택
- 파일 정보 — 모델·버전·크기 (검증 통과 시)
- 검증 상태 — `모델 일치` / `경고` / `불일치`
- `업데이트 시작` — 플래시 시작 (검증 통과 시 활성)
- 진행률 바 — 0% → 100%
- 단계 표시 — 전송 / 검증 / 플래시 / 재부팅

### 절차

- `고급` 탭 > `펌웨어` 섹션
- `펌웨어 정보 읽기` → 현재 모델·버전 확인
- 제조사가 배포한 `.bin` 파일 준비
- `파일 선택` → 자동 무결성 검증
- 모델 일치 확인 후 `업데이트 시작`
- 진행률 100% → 컨트롤러 자동 재부팅
- 재연결 확인

### 모델 매칭

`.bin` 파일 헤더의 모델 식별자와 현재 컨트롤러 모델 비교. 불일치 시 업데이트 차단(방어장치).

> [!warning] 업데이트 중 전원 차단·케이블 제거 절대 금지. 중단 시 컨트롤러가 벽돌(부팅 불가).
> - PC 전원 연결 상태 유지
> - PC 슬립 모드 해제
> - Serial/TCP 연결 견고성 사전 확인
> - 다른 작업 금지

### 실패 시

- 진행률 중단 — 케이블 확인 후 재시도. 반복 실패 시 제조사 지원
- 부팅 반복 — 자동 롤백 미지원 모델일 수 있음. 제조사 문의
- 검증 실패 — 파일 재다운로드 (다운로드 중 손상 가능)
