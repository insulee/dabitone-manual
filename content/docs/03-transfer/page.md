# 페이지메시지

> [!abstract] 컨트롤러에 표시할 페이지 수 송신·기존 페이지 초기화
> `전송` 탭 우측 `페이지메시지` 그룹박스.

### 화면 구성

- `페이지 수` — 드롭다운 (`1` 등)
- `전송` — 선택한 페이지 수 컨트롤러로 송신
- 초기화 대상 — 드롭다운 (`전체` 등)
- `초기화` — 선택 대상 페이지 슬롯 비우기

### 일반 흐름

- `페이지 수` 드롭다운에서 표시할 페이지 수 선택
- `전송` 클릭 → 하단 `로그`에서 송수신 확인
- 페이지 슬롯을 비우려면 초기화 대상 선택 후 `초기화`

> [!warning] `초기화` 실행 시 해당 페이지의 콘텐츠 데이터 삭제. 되돌릴 수 없음.
