# 전광판 켜기/끄기

> [!abstract] 전광판 표출 상태를 즉시 ON 또는 OFF
> `전송` 탭 우측 `전광판 켜기/끄기` 그룹박스.

### 화면 구성

- `켜기` — 표출 ON 송신
- `끄기` — 표출 OFF 송신

### 일반 흐름

- `켜기` 클릭 → 컨트롤러 표출 ON 명령 송신
- `끄기` 클릭 → 컨트롤러 표출 OFF 명령 송신
- 결과는 하단 `로그`에서 확인

> [!info] `끄기`는 LED 표출만 중단. 컨트롤러 전원·통신은 유지되어 원격 `켜기` 가능. 물리 전원 차단은 점검·수리 시에만.
