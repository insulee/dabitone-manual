# 섹션별 효과

> [!abstract] 분할된 섹션 간 효과 동작 방식 송신
> `전송` 탭 우측 `섹션별 효과` 그룹박스.

### 화면 구성

- 효과 모드 드롭다운 — `효과 동시 표출` 등
- `전송` — 선택 모드 적용

### 일반 흐름

- 드롭다운에서 효과 모드 선택
- `전송` → 하단 `로그`에서 송수신 확인

> [!info] 분할 섹션은 [편집](/docs/04-editor/overview) 탭의 `콘텐츠 편집` 다이얼로그에서 `+분할 추가`로 정의.
