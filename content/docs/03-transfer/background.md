# 배경이미지

> [!abstract] 등록된 배경 레이어 항목을 컨트롤러로 송신
> `전송` 탭 우측 `배경이미지` 그룹박스.

### 화면 구성

- 배경 드롭다운 — `사용안함` / 등록된 배경 항목
- `전송` — 선택 배경을 컨트롤러에 적용

### 일반 흐름

- 드롭다운에서 적용할 배경 선택 (`사용안함`이면 배경 제거)
- `전송` → 하단 `로그`에서 송수신 확인

> [!info] 배경 레이어 위에 전면 메시지가 표출. 가독성을 위해 적용 후 실 화면 확인 권장.

> [!info] 새 배경 등록은 [편집](/docs/04-editor/overview) 탭에서 작업.
