# 메시지 전송

> [!abstract] `HEX 프로토콜`·`ASCII 프로토콜` 두 그룹박스로 컨트롤러에 메시지 직접 송신
> `전송` 탭 좌측. 일반 운영은 `HEX 프로토콜`, raw 패킷은 `ASCII 프로토콜`.

### HEX 프로토콜

#### 메시지 설정
- `메시지종류` — `실시간` / `페이지` 라디오
- `섹션` — `0` / `1` / `2` 라디오
- `표시반복` — 드롭다운 (예: `무한반복`)
- `화면전환` — 드롭다운 (예: `즉시전환`)

#### 폰트
- `인코딩/폰트` — `EUC-KR/한글조합형` 등
- `폰트크기` — `16픽셀(기본)` 등
- `폰트그룹` — `폰트그룹 1` ~ `4` (설정 탭에서 등록)

#### 효과
- `입장효과` — 효과 + 방향 드롭다운 2개
- `퇴장효과` — 효과 + 방향 드롭다운 2개
- `효과속도` — 숫자 (작을수록 빠름)
- `유지시간` — 초 단위

#### 표시 영역
- `가로영역(X)` — 시작·끝 픽셀
- `세로영역(Y)` — 시작·끝 픽셀

#### 색상·배경
- `배경이미지` — 사용안함 / 등록된 배경
- `문자색` / `배경색` — 색상 코드 입력
  - `0` 검정 / `1` 빨강 / `2` 초록 / `3` 노랑 / `4` 파랑 / `5` 분홍 / `6` 하늘 / `7` 흰색

#### 메시지 입력
- 단일 라인 텍스트박스 (예: `12ab가나다`)

#### 버튼
- `초기화` — 입력값 리셋
- `ASCII 변환` — HEX 입력값을 ASCII 프로토콜 박스로 변환
- `HEX 전송` — 컨트롤러 송신

### ASCII 프로토콜

- 멀티라인 텍스트박스
- `다중입력` — 여러 줄 일괄 처리
- `패킷복사` — 현재 패킷 클립보드 복사
- `기본값 설정` — 기본 패킷 자동 입력
- `ASCII 전송` — 컨트롤러 송신

### 일반 흐름

- 메시지 입력 박스에 텍스트 작성
- 폰트·효과·색상 설정
- `HEX 전송` 클릭 → 하단 `로그`에서 송수신 확인
- raw 패킷이 필요하면 `ASCII 변환` → ASCII 박스에서 수정 후 `ASCII 전송`

> [!info] `문자색`을 `1234567`로 입력하면 글자별로 빨강·초록·노랑·파랑·분홍·하늘·흰색 순서 적용. 배경색은 단일 자리만.
