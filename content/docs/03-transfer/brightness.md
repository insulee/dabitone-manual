# 밝기

> [!abstract] 전광판 밝기 단일 송신
> `전송` 탭 우측 `밝기` 그룹박스.

### 화면 구성

- 밝기 드롭다운 — `100%` / `75%` / `50%` / `25%` / `5%`
- `전송` — 선택값을 컨트롤러에 즉시 적용
- `고급설정` — 시간대·환경 기반 세부 설정 창 열기

### 일반 흐름

- 드롭다운에서 원하는 밝기 % 선택
- `전송` → 하단 `로그`에서 송수신 확인
- 시간대별 자동 조절이 필요하면 `고급설정` 진입

> [!info] LED 수명을 위해 낮에도 75~90% 권장. 100% 장시간 사용 시 수명 단축.
