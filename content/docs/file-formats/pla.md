# PLA

> **PLA** — **Play List**. 여러 [DAT](/file-formats/dat)·[ANI](/file-formats/ani) 파일을 **시간 순서·조건·반복**과 묶은 플레이리스트. 전광판에서 **전면 콘텐츠 순환**을 담당.

## 어디서 만들어지나

[편집 → 스케줄 그리드](/04-editor/schedule-grid)에서 만듭니다. 우측 파일 목록에서 DAT·ANI를 드래그해 타임라인에 배치.

## 구조

- **항목 목록**: 재생할 DAT·ANI 파일 참조 배열
- **항목별 설정**:
  - 시작 시각·종료 시각
  - 요일·날짜 조건 (예: 평일만, 특정 날짜만)
  - 반복 횟수 또는 시간
  - 표시 효과·속도
- **전체 반복**: 플레이리스트가 끝나면 처음부터

## 어디에 저장되나

```
기본: C:\Users\[사용자]\Documents\DabitOne\Data\
확장자: .PLA
```

## 재생 동작

컨트롤러는 PLA를 로드한 뒤 **자체 실시간 시계**를 기준으로 순차 재생합니다.

- 컨트롤러 시계가 맞아야 시간대별 조건이 의도대로 동작
- 시계가 어긋났다면 [고급 → 시간 설정](/05-advanced/time)에서 동기화

## BGP와의 관계

PLA는 **전면 레이어**, [BGP](/file-formats/bgp)는 **배경 레이어**. 두 레이어가 동시에 재생되며 투명도에 따라 겹쳐 보입니다.

```
[BGP 배경 이미지·패턴 순환]
       ↓ (투명도로 겹침)
[PLA 전면 메시지·안내 순환]
       ↓
[관객이 보는 화면]
```

## 제약

| 항목 | 제약 |
|------|-----|
| 항목 수 | 컨트롤러 메모리에 따라 다름 (보통 수십~수백 개) |
| 참조 파일 | 모든 DAT·ANI 파일도 함께 컨트롤러에 전송돼 있어야 함 |
| 시간 정밀도 | 초 단위 |

> [!info] PLA 파일 자체는 "참조 목록"입니다. 실제 콘텐츠 데이터(DAT·ANI)는 별도로 컨트롤러에 저장되어 있어야 합니다. 전송 시 DabitOne이 자동으로 필요한 파일을 함께 보냅니다.

## 호환성

- 기존 DabitChe PLA 파일 **그대로 호환**
- DabitOne에서 새로 만든 PLA에 신기능(조건식 확장 등)이 있으면 구버전에선 일부 항목이 무시될 수 있음

## 관련 파일 형식

- [DAT](/file-formats/dat) — PLA가 참조하는 프레임/이미지
- [ANI](/file-formats/ani) — PLA가 참조하는 애니메이션
- [BGP](/file-formats/bgp) — 배경 레이어용 (PLA와 동시 재생)
