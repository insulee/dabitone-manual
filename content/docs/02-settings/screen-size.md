# 화면 크기·배열·색상

> [!abstract] 전광판 물리 구성을 DabitOne에 알리는 가장 기본 설정
> `설정` 탭 > `화면 크기` 섹션. 여기 입력값이 모든 편집·전송의 기준.

### 화면 구성

- `가로 모듈 수` — 실물 가로 방향 모듈 개수
- `세로 모듈 수` — 실물 세로 방향 모듈 개수
- `모듈 해상도` — 모듈 라벨 표기값 (16×16 / 32×16 / 32×32 / 64×32)
- `색상 깊이(BPP)` — `1` / `3` / `8` / `24` bit
- `배열 유형` — Normal / Rotate 90·180·270 / Flip H·V
- `적용` — 입력값을 컨트롤러로 전송

### 절차

- `설정` 탭 진입
- 가로·세로 모듈 수 입력
- `모듈 해상도` 선택 → 모듈 라벨 기준
- `색상 깊이` 선택 → 모듈 지원 최대값
- `배열 유형` 선택 → 보통 `Normal`
- `적용` 클릭

### 색상 깊이 (BPP)

- `1 bit` — 단색 (구형 단색 전광판)
- `3 bit` — 7색 + 검정 (구형 3색 전광판)
- `8 bit` — 256색 (중간급)
- `24 bit` — 풀컬러 (현대 풀컬러)

> [!warning] BPP를 모듈 사양보다 높게 잡으면 색상이 깨진다. 모듈 데이터시트 확인 필수.

> [!info] 화면 크기 설정은 컨트롤러 비휘발 메모리에 저장. DabitOne 재설치해도 컨트롤러 값은 유지.
