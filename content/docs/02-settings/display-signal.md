# 표출 신호

> [!abstract] 화면 크기·BPP만으로 색상이 안 맞을 때 추가 조정하는 신호 설정
> `설정` 탭 > `표출 신호` 섹션. 대부분 기본값으로 충분, 특수 모듈·혼합 구성에서만 사용.

### 화면 구성

- `Color Order` — R/G/B 색상 라인 순서 (기본 `RGB`)
- `Scan Order` — 모듈 내부 스캔 방식 (기본 `1/16`)
- `Data Rate` — 신호 속도 (기본 `Normal`)
- `자동 전송 간격` — 주기적 자동 재전송 (초, 기본 `0` = 사용 안 함)
- `Gamma` — 감마 보정값 (기본 `2.2`)

### Color Order

- `RGB` — 표준 (대부분 모듈)
- `RBG` / `BRG` / `BGR` / `GRB` / `GBR` — 특수 모듈 또는 재배선

색상이 바뀌어 나오면 순서를 바꿔 시도.

### Scan Order

- `1/4` — 구형 옥외 모듈
- `1/8` — 중급 옥외 모듈
- `1/16` — 표준 (대부분)
- `1/32` — 고해상도 실내 모듈

잘못 선택 시 가로선이 끊기거나 밝기 불균일.

### 자동 전송 간격

- `0` — 수동 전송 시에만 화면 갱신
- `> 0` 초 — 해당 주기로 DabitOne이 자동 재전송

> [!warning] 자동 전송 간격을 1초처럼 너무 짧게 잡으면 네트워크·컨트롤러 부하. 30초 이상 권장.

> [!info] Gamma는 옥외용 `2.2~2.5`가 일반적. 낮을수록 중간 톤이 밝아진다.
