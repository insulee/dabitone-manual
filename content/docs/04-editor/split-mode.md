# 분할 모드

> [!abstract] 전광판을 여러 섹션으로 나눠 각각 다른 콘텐츠 동시 재생
> `편집` 탭 > `분할 모드`. 예: 상단 공지 + 하단 시계 / 좌측 이미지 + 우측 뉴스 티커.

### 화면 구성

- `분할 레이아웃` — 좌우 2분할 / 상하 2분할 / 4분할 / 상단+하단 2분할 / 사용자 정의
- 섹션 클릭 → 미니 편집기 열림
- `경계선 표시` — 섹션 사이 얇은 선 (체크 해제 권장)
- `저장` — 분할 설정 파일 생성

### 절차

- `편집` 탭 > `분할 모드`
- `분할 레이아웃` 선택
- 각 섹션 클릭 → DAT/ANI/텍스트 배정
- (선택) `사용자 정의`에서 경계선 드래그로 비율 조정
- `Ctrl+S` → 분할 설정 저장
- 섹션별 효과는 [섹션 효과](/docs/03-transfer/section-effect)에서

### 레이아웃 종류

- 좌우 2분할 — `[좌][우]`
- 상하 2분할 — `[상] / [하]`
- 4분할 — `[좌상][우상] / [좌하][우하]`
- 상단+하단 2분할 — `[상] / [좌하][우하]`
- 사용자 정의 — 경계선 자유 드래그

### 제약

- 최대 섹션 수 — 4 (컨트롤러 사양에 따라 더 가능)
- 각 섹션 최소 해상도 — 모듈 1개 이상
- 총 용량 — 섹션 콘텐츠 합이 컨트롤러 메모리 내

> [!warning] 작은 섹션엔 글자 크기 자동 축소 필수. 너무 많은 분할은 가독성 급락.

> [!info] 분할 레이아웃과 섹션 콘텐츠는 컨트롤러 비휘발 메모리에 저장. 재부팅 후 유지.
