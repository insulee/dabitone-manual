# 텍스트 편집

> [!abstract] 전광판 표출용 텍스트 메시지 제작 기본 편집기
> `편집` 탭 > `텍스트` 모드 (편집 탭 진입 시 기본).

### 화면 구성

#### 상단 툴바
- `글꼴` — 내장 폰트 + 업로드된 사용자 폰트
- `크기` — 8/10/12/14/16pt
- `색상` — 텍스트 색상 (ColorPicker)
- `배경색` — 배경 색상 (보통 검정)
- `굵게` / `기울임` — 서체 스타일
- `정렬` — 좌·중·우

#### 본문
- 멀티라인 텍스트 박스
- 글자별 개별 색상 (드래그 후 색상 변경)
- 자동 줄바꿈 또는 수동 `\n`

#### 효과
- `입장 효과` — 좌→우 흐름·점멸·페이드·슬라이드
- `속도` — 느림/중간/빠름 (3~5단계)
- `표시 시간` — 효과 후 유지 시간 (초)

### 절차

- `편집` 탭 > `텍스트` 모드
- 본문 영역에 텍스트 입력
- 글꼴·크기·색상 툴바에서 선택
- (선택) 효과·속도 설정
- `Ctrl+S` → `.DAT` 저장 (효과 포함 시 `.ANI`)
- `전송` 탭 > `메시지`로 송신

### 글자별 색상

- 텍스트 드래그로 선택
- 색상 변경 → 해당 글자만 적용
- 예: `긴급:` 빨강, 나머지는 흰색

### 효과 vs 정적

고정 표시 시 `입장 효과`를 `즉시`로, `표시 시간`을 길게.

> [!info] `Ctrl+Shift+S`로 다른 이름 저장. 사용자 폰트는 [폰트 전송](/docs/02-settings/font)에서 업로드 후 `글꼴` 드롭다운에 노출.
