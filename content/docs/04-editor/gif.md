# GIF 편집

> [!abstract] GIF 애니메이션 직접 제작·편집 (DabitOne 신기능)
> `편집` 탭 > `GIF` 모드. 저장 시 `.ANI`로 변환되어 전송.

### 화면 구성

#### 좌측 — 타임라인 패널
- 프레임 썸네일 — 전체 프레임 목록
- `+` — 빈 프레임 추가
- `복사` — 선택 프레임 복사
- `삭제` — 선택 프레임 제거
- 드래그 재배치 — 프레임 순서 이동

#### 가운데 — 편집 캔버스
- 이미지 편집기와 동일 도구 (그리기·텍스트·이미지)

#### 우측 — 속성 패널
- `프레임 딜레이` — 현재 프레임 표시 시간 (ms)
- `반복 횟수` — 전체 반복 (`0` = 무한)
- `전환 효과` — 프레임 간 페이드·인스턴트
- `총 길이` — 전체 재생 시간

#### 하단 — 재생 컨트롤
- ▶ 재생 / ⏸ 일시정지 / ⏹ 정지
- 재생 속도 배율 (0.5x / 1x / 2x)

### 절차

- `편집` 탭 > `GIF` 모드
- `새로 만들기` 또는 `GIF 불러오기`
- 좌측 타임라인에서 프레임 선택 → 가운데 캔버스 편집
- 각 프레임 `딜레이` 설정
- `반복 횟수` 입력
- `Ctrl+S` → `.ANI` 또는 `.gif` 내보내기

### 외부 GIF 불러오기

표준 GIF 불러오면 각 프레임이 내부 편집 프레임으로 변환. 이후 일반 이미지처럼 수정.

### 용량 관리

- 프레임 수 ≤ 50 권장
- 해상도는 전광판 기준 (불필요하게 키울 필요 없음)

> [!info] 저장 포맷: `.ANI`(컨트롤러 전송용) / `.gif`(외부 시스템·문서용).
