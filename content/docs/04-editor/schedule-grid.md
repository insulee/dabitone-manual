# 스케줄 그리드

> [!abstract] 여러 DAT·ANI를 시간 순서로 배치해 PLA 또는 BGP 제작
> `편집` 탭 > `스케줄` 모드. 시간 기반 자동 표출 플레이리스트.

### 화면 구성

#### 좌측 — 타임라인 그리드
- 24시간 또는 주 단위 시간 축
- 각 항목은 색상 블록, 드래그로 길이·위치 조정

#### 우측 — 파일 탐색기
- 폴더 트리 — DabitOne 프로젝트 폴더
- 파일 목록 — 선택 폴더의 DAT/ANI
- 미리보기 — 선택 파일 미리보기

#### 하단 — 항목 속성
- `시작 시각` / `종료 시각` — 재생 시간 범위
- `요일` — 월~일 복수 선택
- `날짜` — 특정 날짜만 (공휴일 등)
- `반복` — 시간 내 반복 횟수
- `효과` — 입장·퇴장

### 절차

- `편집` 탭 > `스케줄` 모드
- `새로 만들기` → PLA 또는 BGP 모드 선택
- 우측 파일 탐색기에서 DAT/ANI를 타임라인에 드래그
- 각 항목 클릭 → 시간·요일·반복 설정
- `Ctrl+S` → `.PLA` 또는 `.BGP`

### PLA vs BGP

- `PLA` — 전면 메시지 순환 (해당 시간에 메시지 표출)
- `BGP` — 배경 이미지 순환 (배경 레이어에 이미지)

저장 시 확장자로 결정.

### 조건식 예시

- 매주 월~금 09:00-18:00에 메시지 A
- 12/25, 1/1에만 메시지 B
- 짝수 날짜에만 메시지 C

> [!info] `시뮬레이션 재생`으로 전체 스케줄 빠르게 검증 가능. 현장 가기 전 필수.

> [!warning] PLA에 포함된 DAT/ANI는 자동 참조만. 함께 전송하려면 `전송` 탭에서 일괄 전송 옵션 사용.
