# 이미지 편집

> [!abstract] 외부 이미지를 전광판 해상도·색상에 맞춰 DAT로 변환
> `편집` 탭 > `이미지` 모드. 로고·아이콘·사진 표출에 사용.

### 화면 구성

#### 툴바
- `불러오기` — 파일 탐색 (또는 드래그 앤 드롭)
- `크롭` — 이미지 일부만 잘라 사용
- `중앙 정렬` — 캔버스 중앙으로
- `크기 조정` — 10~300% 슬라이더
- `회전` — 0/90/180/270도
- `플립` — 좌우·상하 뒤집기

#### 팔레트 편집 (저BPP)
- 1bit/3bit 모드에서 어떤 색을 켜짐으로 할지 수동 조정

#### 미리보기
- 전광판 실제 크기로 실시간 렌더

### 절차

- `편집` 탭 > `이미지` 모드
- `불러오기` 또는 드래그 앤 드롭
- 크기·위치 조정 (드래그·숫자 입력)
- (선택) 색상·팔레트 편집
- (선택) 효과 설정
- `Ctrl+S` → `.DAT`

### 색상 깊이 변환

24비트 원본을 3/8비트로 축약 시 디더링 적용 여부 선택:

- 디더링 ON — 그라데이션 부드럽지만 패턴 보일 수 있음
- 디더링 OFF — 색상 밴드 뚜렷, 깔끔

### 투명 PNG

투명 배경은 검정 또는 지정 색으로 채워짐. 배경색 선택 가능.

> [!info] 효과 포함 저장 시 자동으로 `.ANI`. 편집 원본은 DabitOne 내부 포맷으로 별도 저장(재편집 가능).
