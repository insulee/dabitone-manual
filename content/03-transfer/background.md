# 배경 이미지 전송


> 경로: **[전송] → 배경 이미지 섹션**

## 이 화면의 역할

전광판의 **배경 레이어**에 이미지 또는 스케줄([BGP](/file-formats/bgp))을 전송합니다. 전면 메시지([PLA](/file-formats/pla))가 그 위로 표출됩니다.

## 기본 사용 — 단일 배경 이미지

1. `파일 선택` — `.DAT` 또는 `.BGP` 선택
2. `투명도` 조정 (0~100%)
3. **[전송]** → 배경 레이어에 적용

## 기본 사용 — 배경 스케줄

여러 배경 이미지를 순환시키려면:

1. [편집 → 스케줄 그리드](/04-editor/schedule-grid)에서 `.BGP` 파일 제작
2. 이 화면에서 `.BGP` 선택 후 전송

## 화면 구성

| 요소 | 유형 | 설명 |
|------|-----|-----|
| `파일 선택` | Button | `.DAT` / `.BGP` 선택 |
| `투명도` | Slider | 0 (완전 투명) ~ 100 (불투명) |
| `블렌드 모드` | ComboBox | Normal / Multiply / Screen (컨트롤러 지원 시) |
| 미리보기 | Image | 현재 배경 미리보기 |
| **[전송]** | Button | 업로드 |
| **[제거]** | Button | 배경 레이어 비우기 |

## 주요 옵션

### 투명도 권장값

| 용도 | 투명도 |
|------|-------|
| 은은한 브랜드 배경 | 20~40% |
| 반투명 패턴 | 40~60% |
| 선명한 배경 (전면 가독성 낮음) | 70~100% |

> [!warning] 투명도를 너무 높이면 전면 메시지가 읽기 어려워집니다. 사전에 가독성 테스트 필수.

### 블렌드 모드

모든 컨트롤러가 지원하지 않을 수 있음. 지원 여부는 펌웨어 사양 확인.

### 제거

배경을 비우면 전광판 배경이 **단색 검정** 또는 컨트롤러 기본값으로 돌아감.

## 설정 영속성

배경 레이어는 컨트롤러 비휘발 메모리에 저장. 재부팅 후에도 유지.
