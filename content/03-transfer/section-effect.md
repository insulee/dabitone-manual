# 섹션 효과

> 경로: **[전송] → 섹션 효과 섹션** (분할 모드 사용 시)

## 이 화면의 역할

[분할 모드](/04-editor/split-mode)로 나눈 **각 섹션에 서로 다른 효과**를 적용. 예를 들어 좌측 섹션은 고정, 우측 섹션은 흐름 효과.

## 기본 사용

1. `섹션 선택` — 분할 모드에서 정의한 섹션 번호 (예: 1, 2, 3)
2. 해당 섹션의 `입장 효과`, `퇴장 효과`, `속도` 설정
3. **[적용]** → 전송

## 화면 구성

| 요소 | 유형 | 설명 |
|------|-----|-----|
| `섹션 번호` | ComboBox | 분할된 섹션 선택 |
| `입장 효과` | ComboBox | Fade / Slide / Wipe / 즉시 등 |
| `퇴장 효과` | ComboBox | 입장과 동일 옵션 |
| `속도` | Slider | 효과 지속 시간 (ms) |
| `표시 시간` | NumericUpDown | 효과 완료 후 표시 유지 시간 (초) |
| 미리보기 | 영역 | 효과 시뮬레이션 |
| **[적용]** | Button | 전송 |

## 주요 옵션

### 효과 종류

- **Fade** — 투명도로 부드럽게 등장/사라짐
- **Slide (Left/Right/Up/Down)** — 지정 방향으로 슬라이드
- **Wipe (Horizontal/Vertical)** — 가운데서 양쪽으로 펼쳐짐
- **Zoom** — 중앙에서 확대/축소
- **즉시** — 효과 없이 바로 표시

### 속도 권장값

| 상황 | 속도 (ms) |
|------|----------|
| 빠른 전환 (광고) | 200~400 |
| 중간 (일반 공지) | 500~800 |
| 천천히 (차분한 정보) | 1000~1500 |

너무 빠르면 가독성 저하, 너무 느리면 지루함.

### 섹션별 효과 충돌 주의

여러 섹션에 서로 다른 효과가 동시에 재생될 때 **시각적으로 산만**할 수 있음. 보통 주 섹션은 효과, 보조 섹션은 고정으로 가는 게 좋음.

## 설정 영속성

섹션 효과 설정은 컨트롤러에 저장됩니다.
