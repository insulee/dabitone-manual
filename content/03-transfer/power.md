# 켜기·끄기 스케줄


> 경로: **[전송] → 켜기·끄기 섹션**

## 이 화면의 역할

전광판 표출을 **즉시 껐다 켜거나**, 주야간·요일별 자동 ON/OFF 스케줄을 설정.

## 기본 사용 — 수동

1. **[ON]** 또는 **[OFF]** 버튼 클릭 → 즉시 반영
2. 상태 표시가 "표출 중" / "꺼짐"으로 변경

## 기본 사용 — 자동 스케줄

1. `자동 스케줄` 체크
2. 요일별 ON/OFF 시간 입력 (예: 평일 06:00 ON / 22:00 OFF)
3. **[전송]** → 컨트롤러에 저장

## 화면 구성

| 요소 | 유형 | 설명 |
|------|-----|-----|
| 현재 상태 | 표시 | "표출 중" / "꺼짐" |
| **[ON]** / **[OFF]** | Button | 즉시 전환 |
| `자동 스케줄` | CheckBox | 스케줄 사용 |
| 요일별 테이블 | DataGrid | 월~일 각각 ON 시각·OFF 시각 |
| **[전송]** | Button | 스케줄 업로드 |

## 주요 옵션

### OFF와 전원 차단의 차이

- **OFF** (이 화면): LED 표출만 중단, 컨트롤러 전원·통신은 유지 → 원격으로 다시 ON 가능
- **전원 차단**: 물리적 전원 OFF → 원격 제어 불가

OFF가 더 유용. 전원 차단은 점검·수리 시에만.

### 요일별 예시

| 요일 | ON | OFF |
|------|-----|-----|
| 월~금 | 06:00 | 22:00 |
| 토 | 08:00 | 23:00 |
| 일 | 09:00 | 21:00 |

상가·업무용은 영업 시간에 맞춰, 공공·안내용은 하루 종일 ON하되 심야는 밝기 낮추기(위 참조).

### 휴일 예외

현재 UI는 요일별만 지원. 특정 휴일·공휴일 예외 처리는 v1.2.0에 예정.

## 설정 영속성

컨트롤러 비휘발 메모리 저장 + 컨트롤러 시계 기반 자동 실행.
