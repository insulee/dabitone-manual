title: 빠른 시작
---

현장에서 자주 하는 작업을 단계별로 안내합니다. 각 페이지는 "무엇을 준비하고, 어떤 버튼을 어떤 순서로 누르며, 성공하면 무엇이 보여야 하는지"에 집중합니다.

## 시나리오

1. [[quickstart/01-first-connection|컨트롤러 최초 연결]] — Serial·TCP·UDP 중 선택하고 연결 확인
2. [[quickstart/02-screen-size|화면 크기 설정]] — 가로·세로 모듈 수와 색상 깊이 전송
3. [[quickstart/03-send-message|첫 메시지 전송]] — 텍스트 입력 후 바로 표출하기
4. [[quickstart/04-edit-image|이미지 편집·전송]] — BMP/PNG/JPG를 DAT로 변환해 보내기
5. [[quickstart/05-gif-editor|GIF 편집]] — 내장 편집기로 동영상 제작
6. [[quickstart/06-schedule-pla|스케줄 편집 (PLA)]] — 여러 메시지를 순차 재생
7. [[quickstart/07-background-bgp|배경 스케줄 (BGP)]] — 배경화면 순환
8. [[quickstart/08-firmware|펌웨어 업데이트]] — 컨트롤러 펌웨어 갱신

## 처음 세 단계만 해도 됩니다

연결(01) → 화면 크기(02) → 메시지(03)까지만 하면 전광판에 원하는 글자를 띄울 수 있습니다. 나머지는 필요해질 때 읽으세요.
