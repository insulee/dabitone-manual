# 보드 기능 설정

> 경로: **[고급] → 보드 기능 섹션**
> 
> ⚠️ **전문가용**. 잘못 설정하면 컨트롤러가 응답하지 않거나 화면이 깨질 수 있습니다.

## 이 화면의 역할

컨트롤러 보드의 **저수준 하드웨어 옵션**을 조정. 일반 사용자는 거의 건드릴 일 없음. 설치 초기 또는 제조사 지원 요청 시에만 사용.

## 기본 사용

> [!warning] 현재 값 **반드시 메모** 후 변경하세요. 잘못 바꿔 응답 없음이 되면 공장 초기화가 필요할 수 있습니다.

1. 각 항목의 현재 값 확인
2. 필요한 항목만 변경
3. **[적용]** → 컨트롤러 재부팅 후 반영

## 화면 구성

| 요소 | 유형 | 설명 |
|------|-----|-----|
| `디버깅 모드` | CheckBox | 내부 로그 출력 활성화 |
| `BH1 모드` | ComboBox | BH1 핀 동작 (입력/출력/특수) |
| `J2 (외부 센서)` | ComboBox | J2 커넥터 용도 (온도/조도/미사용) |
| `J3 (외부 장치)` | ComboBox | J3 커넥터 용도 |
| `직렬 보드레이트` | ComboBox | Serial 포트 속도 (9600~921600) |
| `네트워크 보드레이트` | ComboBox | 이더넷 속도 (10/100 Mbps) |
| `모듈 스캔 주파수` | NumericUpDown | Hz 단위 |
| **[공장 초기화]** | Button | 모든 보드 설정을 공장 기본값으로 |

## 주요 옵션

### 디버깅 모드

활성화 시 컨트롤러가 상세 로그를 출력 (Serial 모니터로 확인 가능). 문제 조사 시 제조사가 요청. 일반 운영에선 끔.

### BH1 / J2 / J3 커넥터

제품별로 역할이 다름. 설명서 확인 후 설정. 일반적으로:

- **BH1**: 추가 제어 신호
- **J2**: 외부 센서 (밝기 센서 등으로 자동 밝기 조절)
- **J3**: 외부 장치 (다른 컨트롤러와 동기화 등)

### 모듈 스캔 주파수

화면 새로고침 빈도. 너무 낮으면 깜빡임 인식, 너무 높으면 밝기 저하. 기본값 추천.

### 공장 초기화

> [!warning] **보드 설정만 초기화**되며 저장된 메시지·스케줄은 유지됩니다. 단 해당 하드웨어 설정에 의존한 동작은 리셋됨.

## 제약

- 변경 후 **컨트롤러 재부팅** 필수
- 일부 항목은 변경 시 **일시적으로 연결 끊김** 발생 (특히 보드레이트)

## 관련

- 제조사 기술 문서 (하드웨어별 세부 스펙)
- 문제 해결 시 제조사 지원 요청 권장
