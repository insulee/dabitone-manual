# 테마

> 경로: **[고급] → 테마 섹션**

## 이 화면의 역할

DabitOne 앱의 **UI 테마**(라이트·다크)를 선택. 전광판 표출과는 무관하며, 사용자 편의용 설정입니다.

## 기본 사용

라디오 버튼에서 원하는 테마 선택:

- **라이트** — 밝은 배경, 진한 텍스트
- **다크** — 어두운 배경, 밝은 텍스트 (야간 작업에 적합)
- **시스템 연동** — Windows 설정의 라이트/다크 모드를 자동 반영

즉시 전환됩니다. 재시작 불필요.

## 화면 구성

| 요소 | 유형 | 설명 |
|------|-----|-----|
| `테마` | RadioButton | 라이트 / 다크 / 시스템 연동 |
| `액센트 색상` | ColorPicker | 강조 색상 변경 (기본 파랑 `#2563EB`) |
| 미리보기 | 영역 | 선택 테마가 바로 반영된 UI 스와치 |

## 주요 옵션

### 시스템 연동 권장 이유

Windows 10/11은 낮·밤 자동 전환(또는 수동 전환)이 가능합니다. 시스템 연동을 켜면 DabitOne도 함께 바뀌어 **야간 작업 눈 피로 감소**.

### 액센트 색상

기본 파랑 외에 팀·브랜드 색상으로 바꾸고 싶다면 여기서. UI의 버튼·링크·강조 영역에 반영됨.

> [!info] 액센트 색상을 너무 밝거나 어둡게 하면 가독성이 떨어집니다. 미리보기에서 충분히 확인 후 적용.
