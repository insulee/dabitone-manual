# 밝기 설정

> 경로: **[전송] → 밝기 섹션**

## 이 화면의 역할

전광판의 **밝기**를 설정합니다. 수동 5단계(100/75/50/25/5%) 또는 시간대별 자동 조절.

## 기본 사용 — 수동 밝기

1. 라디오 버튼에서 `100%` / `75%` / `50%` / `25%` / `5%` 중 하나 선택
2. **[적용]** → 즉시 반영

## 기본 사용 — 자동 밝기 스케줄 (고급)

1. `자동 조절` 체크
2. 시간대별 밝기 테이블 편집 (예: 00:00-06:00=5%, 06:00-20:00=100%, 20:00-24:00=50%)
3. **[전송]** → 컨트롤러에 스케줄 저장

## 화면 구성

| 요소 | 유형 | 설명 |
|------|-----|-----|
| 밝기 레벨 | RadioButton (5개) | 100·75·50·25·5% |
| `자동 조절` | CheckBox | 시간대별 스케줄 사용 |
| 시간대 테이블 | DataGrid | 시작시간·종료시간·밝기% |
| **[적용]** / **[전송]** | Button | 설정 적용 |

## 주요 옵션

### 왜 5% 단계까지 있나

심야 운영 시 주변 주민에게 **빛 공해** 최소화를 위해. 법규상 특정 시간에 밝기 제한이 있는 지역도 있음.

### 자동 조절 권장 설정

| 시간대 | 밝기 |
|-------|------|
| 00:00 - 06:00 | 5~25% (심야) |
| 06:00 - 09:00 | 50~75% (아침) |
| 09:00 - 18:00 | 100% (낮) |
| 18:00 - 22:00 | 75% (저녁) |
| 22:00 - 24:00 | 25~50% (야간) |

정확한 값은 **주변 환경·조도**에 맞춰 조정.

### LED 수명과 밝기

밝기를 100%로 장시간 사용하면 LED 수명 단축. **낮에도 75~90%** 사용이 수명·가독성 양쪽에 유리.

## 설정 영속성

수동·자동 모두 컨트롤러에 저장. 컨트롤러 시계에 맞춰 자동 스케줄 동작.
