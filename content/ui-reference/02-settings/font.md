# 폰트 전송

![폰트 전송 UI](/assets/screens/reference/settings-font.png)

> 경로: **[설정] → 폰트 섹션**

## 이 화면의 역할

컨트롤러 내부 폰트 슬롯에 **새 폰트 파일(FNT)**을 업로드. 대부분은 공장 출하 기본 폰트면 충분하지만, 로고·특수 기호용 커스텀 폰트를 쓸 때 필요.

## 기본 사용

1. `파일 선택` — PC의 `.FNT` 파일 선택
2. `슬롯 번호` — 저장할 컨트롤러 폰트 슬롯 (1~4, 컨트롤러마다 다름)
3. **[전송]** → 업로드 진행률 표시 후 완료
4. [편집 → 텍스트](/ui-reference/04-editor/text) 글꼴 드롭다운에서 업로드한 폰트 확인

## 화면 구성

| 요소 | 유형 | 설명 |
|------|-----|-----|
| `파일 선택` | Button + 경로 표시 | `.FNT` 파일 선택 대화상자 |
| `슬롯 번호` | ComboBox | 1 / 2 / 3 / 4 (컨트롤러 사양에 따라) |
| `현재 슬롯 정보` | 표시 | 해당 슬롯에 이미 들어 있는 폰트 이름 |
| **[전송]** | Button | 업로드 시작 |
| 진행률 바 | ProgressBar | 업로드 진행 표시 |

## 주요 옵션

### 슬롯 관리

- **슬롯 1**: 보통 기본 한글 폰트 (덮어쓰면 한글 깨질 수 있음, 주의)
- **슬롯 2~4**: 추가 폰트용

> [!warning] 기본 폰트 슬롯을 덮어쓰기 전에 **원본 백업**을 해두세요. 기본 폰트가 사라지면 기존 저장된 메시지가 깨질 수 있습니다.

### 파일 크기

FNT 파일 1개당 수십~수백 KB. 컨트롤러 총 폰트 저장공간이 제한적이므로 큰 폰트를 여러 개 업로드하면 다른 콘텐츠 저장공간이 줄 수 있음.

## 설정 영속성

폰트는 **컨트롤러 비휘발 메모리에 저장**됩니다. 업로드 한 번으로 재부팅 후에도 유지.
