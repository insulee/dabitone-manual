# 설정

> 경로: 좌측 사이드바 **[설정]** 탭

## 이 화면의 역할

컨트롤러의 **기본 운영 설정**을 한 화면에 모은 창. 설치 직후 한 번 맞추고 나면 자주 건드리지 않지만, 교체·재배치·펌웨어 갱신 후에는 다시 확인합니다.

## 주요 섹션

- [[ui-reference/02-settings/screen-size|화면 크기]] — 가로/세로 모듈 수, 모듈 해상도, 배열 방향
- [[ui-reference/02-settings/display-signal|표출 신호]] — 색상 깊이·스캔 방식·새로고침 주기
- [[ui-reference/02-settings/font|폰트 전송]] — 컨트롤러에 사용자 폰트 업로드

## 설정 영속성

> [!info] 이 화면에서 바꾼 값은 **[적용]** 또는 **[전송]** 버튼으로 컨트롤러에 즉시 반영됩니다. 반영된 값은 컨트롤러 쪽 비휘발 메모리에 저장되어 재부팅 후에도 유지.
