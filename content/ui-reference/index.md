title: UI 레퍼런스

DabitONe 메인 화면의 **5그룹 사이드바 순서**대로 각 기능을 설명합니다. "이 버튼이 뭐예요?"를 찾을 때 사용하세요.

## 그룹

1. **[[ui-reference/01-communication/index|통신]]** — Serial, Client/Server TCP, UDP, BLE, MQTT, dbNet
2. **[[ui-reference/02-settings/index|설정]]** — 화면 크기·배열·색상, 표출신호, 폰트 전송
3. **[[ui-reference/03-transfer/index|전송]]** — 메시지(ASCII/HEX), 페이지, 밝기, 켜기·끄기, 배경이미지, 섹션 효과
4. **[[ui-reference/04-editor/index|편집]]** — 스케줄 그리드, 텍스트·이미지·GIF 편집, 분할 모드
5. **[[ui-reference/05-advanced/index|고급]]** — 시간 설정, 보드기능설정, 펌웨어, 테마

## 빠른 시작과의 차이

[[quickstart/index|빠른 시작]]이 "이 작업은 어떻게?"에 답한다면, UI 레퍼런스는 "이 버튼은 무엇?"에 답합니다. 두 트랙은 서로를 가리키는 링크로 연결되어 있습니다.
