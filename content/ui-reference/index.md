# UI 레퍼런스

DabitOne 메인 화면의 **5 그룹 사이드바 순서**대로 각 기능을 설명합니다. "이 버튼이 뭐예요?"를 찾을 때 사용하세요.

## 체험 투어도 있습니다

각 기능을 직접 눌러가며 배우고 싶다면 **[/tour/](/tour/)** 에서 인터랙티브 투어를 확인하세요. 레퍼런스와 투어는 서로 링크로 연결되어 필요할 때 오갈 수 있습니다.

## 5 그룹

1. **[[ui-reference/01-communication/index|통신]]** — Serial, Client/Server TCP, UDP, BLE, MQTT, dbNet
2. **[[ui-reference/02-settings/index|설정]]** — 화면 크기·배열·색상, 표출신호, 폰트 전송
3. **[[ui-reference/03-transfer/index|전송]]** — 메시지(ASCII/HEX), 페이지, 밝기, 켜기·끄기, 배경이미지, 섹션 효과
4. **[[ui-reference/04-editor/index|편집]]** — 스케줄 그리드, 텍스트·이미지·GIF 편집, 분할 모드
5. **[[ui-reference/05-advanced/index|고급]]** — 시간 설정, 보드기능설정, 펌웨어, 테마

## 투어와 레퍼런스의 역할 분담

| 상황 | 어디로 | 왜 |
|------|-------|-----|
| "새 전광판에 처음 연결하는 중" | [/tour/](/tour/) | 전체 흐름을 눈으로 따라가기 |
| "[연결 테스트] 버튼은 뭘 하나" | [[ui-reference/01-communication/index|UI 레퍼런스 > 통신]] | 특정 컨트롤 의미 조회 |

| "연결 테스트 응답 없음이 뜸" | [[troubleshooting/01-connection|문제 해결 > 연결]] | 증상 → 원인 → 조치 |
