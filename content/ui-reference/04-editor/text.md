# 텍스트 편집

> 경로: **[편집] → 텍스트 모드** (편집 탭 진입 시 기본)

## 이 화면의 역할

전광판에 표출할 **텍스트 메시지**를 만드는 가장 기본 편집기. 입력한 글자를 [[file-formats/dat|DAT]] 포맷으로 저장해 전송합니다.

## 기본 사용

1. 편집기 가운데 입력 영역에 텍스트 타이핑
2. 글꼴·크기·색상 툴바에서 선택 (실시간 미리보기)
3. (선택) 효과·속도 설정
4. `Ctrl+S` 또는 **[저장]** → `.DAT` 파일 생성
5. [[ui-reference/03-transfer/message|전송]] 탭으로 이동해 컨트롤러로 송신

## 화면 구성

### 상단 툴바

| 요소 | 유형 | 설명 |
|------|-----|-----|
| `글꼴` | ComboBox | 내장 폰트 + [[ui-reference/02-settings/font|업로드된 사용자 폰트]] |
| `크기` | ComboBox | 8pt / 10pt / 12pt / 14pt / 16pt (폰트 기준) |
| `색상` | ColorPicker | 텍스트 색상 |
| `배경색` | ColorPicker | 배경 색상 (보통 검정) |
| `굵게` / `기울임` | ToggleButton | 서체 스타일 |
| `정렬` | RadioButton | 좌·중·우 |

### 본문 입력 영역

- 멀티라인 텍스트 박스
- **글자별 개별 색상** 지정 가능 (드래그 선택 후 색상 변경)
- 줄바꿈 지원 (해상도에 따라 자동 줄바꿈 또는 수동 `\n`)

### 효과 섹션

| 요소 | 설명 |
|------|------|
| `입장 효과` | 좌→우 흐름 / 점멸 / 페이드 / 슬라이드 등 |
| `속도` | 느림·중간·빠름 (3~5단계) |
| `표시 시간` | 효과 끝난 후 몇 초 유지할지 |

### 미리보기

하단 또는 우측에 **실제 전광판 해상도**로 렌더된 미리보기가 실시간 갱신.

## 주요 옵션

### 글자별 색상

DabitOne에서 강화된 기능. 텍스트를 드래그로 선택하고 색상을 바꾸면 **해당 글자만 색상 적용**. 예: "긴급: 현재 공사 중" → "긴급:"만 빨강, 나머지는 흰색.

### 자동 줄바꿈

해상도보다 텍스트가 길면 자동으로 다음 줄로. 수동으로 강제 줄바꿈하려면 Enter.

### 효과 vs 정적

효과 없이 **고정 표시**하고 싶다면 `입장 효과`를 "즉시"로, `표시 시간`을 길게. 광고·공지 등에서 유용.

## 저장·내보내기

- `Ctrl+S` — DAT 파일 저장
- `Ctrl+Shift+S` — 다른 이름으로 저장
- 효과가 포함된 경우 자동으로 [[file-formats/ani|ANI]]로 저장
