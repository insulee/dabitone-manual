# 스케줄 그리드

> 경로: **[편집] → 스케줄 모드**

## 이 화면의 역할

여러 [[file-formats/dat|DAT]]·[[file-formats/ani|ANI]] 파일을 **시간 순서로 배치**해 플레이리스트([[file-formats/pla|PLA]]) 또는 배경 스케줄([[file-formats/bgp|BGP]])을 제작.

## 기본 사용

1. **[새로 만들기]** — PLA 또는 BGP 모드 선택
2. 우측 파일 탐색기에서 DAT/ANI 파일을 **타임라인에 드래그**
3. 각 항목 클릭 → 시간·조건 설정
4. **[저장]** → `.PLA` 또는 `.BGP`

## 화면 구성

### 좌측: 타임라인 그리드

```
 0:00    3:00    6:00    9:00   12:00   15:00   18:00   21:00
 │       │       │       │       │       │       │       │
 ├───[item1]─────┼──[item2]──────┼───────[item3]──────────┤
```

- 시간 축은 24시간 또는 주 단위
- 각 항목은 색상 블록, 드래그로 길이·위치 조정

### 우측: 파일 탐색기

| 요소 | 설명 |
|------|-----|
| 폴더 트리 | DabitONe 프로젝트 폴더 브라우저 |
| 파일 목록 | 선택 폴더의 DAT·ANI 파일들 |
| 미리보기 | 선택한 파일의 미리보기 |

### 하단: 항목 속성

선택된 타임라인 항목의 속성:

| 속성 | 설명 |
|------|-----|
| `시작 시각` / `종료 시각` | 재생 시간 범위 |
| `요일` | 월~일 중 선택 (복수 가능) |
| `날짜` | 특정 날짜만 (예: 공휴일) |
| `반복` | 해당 시간 내 반복 횟수 |
| `효과` | 입장·퇴장 |

## 주요 옵션

### PLA vs BGP 모드

- **PLA**: 전면 메시지 순환 — 해당 시간에 이 메시지를 표출
- **BGP**: 배경 이미지 순환 — 배경 레이어에 이 이미지를 깔기

모드는 파일 저장 시 확장자로 결정됨.

### 조건식

각 항목에 **논리 조건**을 걸 수 있음. 예:

- "매주 월~금 09:00-18:00에 메시지 A 표출"
- "특정 휴일(12/25, 1/1)에만 메시지 B"
- "짝수 날짜에만 메시지 C"

### 프리뷰

**[시뮬레이션 재생]** 버튼으로 전체 스케줄을 빠르게 돌려 볼 수 있음. 현장 가기 전 검증.

## 저장·내보내기

- `Ctrl+S` — PLA 또는 BGP 파일 저장
- 스케줄에 포함된 DAT·ANI 파일은 **자동 참조**만 — 함께 전송하려면 [[ui-reference/03-transfer/message|전송 탭]]에서 일괄 전송 옵션
