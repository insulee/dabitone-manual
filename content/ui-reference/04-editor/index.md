# 편집

> 경로: 좌측 사이드바 **[편집]** 탭

## 이 화면의 역할

전광판에 표출할 콘텐츠를 **만드는** 작업 공간. DabitOne에서 가장 크게 달라진 영역 — 드래그 앤 드롭, 실시간 미리보기, 내장 GIF 편집기 등이 새롭게 추가됐습니다.

## 편집 모드

- [[ui-reference/04-editor/text|텍스트 편집]] — 글꼴·크기·색상·효과·속도
- [[ui-reference/04-editor/image|이미지 편집]] — BMP/PNG/JPG 가져와 DAT 변환
- [[ui-reference/04-editor/gif|GIF 편집]] — 프레임 타임라인 기반 애니메이션 제작 (**신기능**)
- [[ui-reference/04-editor/schedule-grid|스케줄 그리드]] — 시간대별 콘텐츠 배치 (PLA)
- [[ui-reference/04-editor/split-mode|분할 모드]] — 화면을 영역으로 나눠 각각 다른 콘텐츠

## 공통 워크플로우

1. 편집 모드 선택
2. 콘텐츠 작성·조정
3. 실시간 미리보기 확인
4. 저장 → 자동으로 컨트롤러 포맷(DAT/ANI/PLA/BGP 등)으로 변환
5. [[ui-reference/03-transfer/index|전송]] 탭에서 컨트롤러로 전송

## 파일 포맷

편집 결과물이 저장되는 포맷은 [[file-formats/dat|파일 포맷]] 페이지 참조.
