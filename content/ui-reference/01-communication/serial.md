# Serial

> 경로: **[통신] → Serial 그룹박스**

## 기본 사용

1. `Serial` 그룹박스 헤더의 라디오 버튼 클릭 → 그룹 내부 컨트롤 활성화
2. `포트` 드롭다운에서 COM 번호 선택
3. `속도` 드롭다운에서 컨트롤러 펌웨어와 일치하는 값 선택 (기본 115200)
4. (RS-485 배선이면) `RS-485 Address` 체크 후 주소값 선택
5. 하단 **[연결 테스트]** 클릭

## 화면 구성

| 요소 | 유형 | 설명 |
|------|------|------|
| `포트` | ComboBox | 사용 가능한 COM 포트 목록 (USB-to-Serial 어댑터 포함). Windows가 인식한 것만 나옴 |
| `속도` | ComboBox | 9600 / 19200 / 38400 / 57600 / **115200**(기본) / 230400 / 921600 |
| `RS-485 Address` | CheckBox | RS-485 다중 장치 배선 시 체크. 체크 해제 시 주소 필드 비활성화 |
| 주소값 | ComboBox | RS-485 체크 시 활성화. 1~255 범위 |
| `[속도 찾기]` | Button | 현재 포트에서 여러 속도를 순차 시도해 통신되는 속도 자동 탐색 |
| `[장치관리자]` | Button | Windows 장치관리자를 바로 엶 (새 어댑터 인식 확인용) |

## 주요 옵션

### 속도 (baud rate)

컨트롤러 펌웨어 쪽 설정과 **정확히** 일치해야 합니다. 기본값 115200은 비교적 최근 펌웨어의 기본값이며, 레거시 장비는 9600 또는 38400이 흔합니다.

- 값을 모를 때: **[속도 찾기]** 버튼 사용
- RS-485 환경에서 장비 수가 많을수록 낮은 속도가 안정적
- 921600 등 고속은 짧은 거리·양호한 케이블에서만 사용

### RS-485 Address

RS-485는 한 버스에 여러 장비가 연결되는 환경에서 사용. 각 컨트롤러의 주소값(DIP 스위치 또는 설정)과 일치하는 값을 선택.

## 설정 영속성

> [!info] 선택한 포트·속도·RS-485 주소는 `AppSettings`에 자동 저장됩니다. 같은 COM 포트·속도 조합으로 다음 실행 시 기본값으로 복원.
