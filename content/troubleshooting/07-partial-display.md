# 일부 모듈만 표출·나머지 블랙

전체 전광판이 여러 LED 모듈로 구성되어 있는데, 일부만 메시지가 나오고 나머지가 검게 꺼진 채로 있는 경우.

## 증상

- 좌측 절반만 표시, 우측은 검정
- 상단 1줄만 표시, 나머지 행 검정
- 모듈 단위로 정사각형 영역이 꺼짐
- 전체 화면이 조각난 것처럼 보임

## 원인

1. **화면 크기 설정 불일치** — DabitOne의 가로·세로 모듈 수가 실제 배선과 다름
2. **모듈 전원 결선 문제** — 일부 모듈이 전원·신호를 못 받음
3. **캐스케이드 배선 끊김** — 모듈 간 신호 케이블(Hub-to-Hub) 일부 접속 불량
4. **Scan Order 오류** — 모듈 배치 순서(좌→우 vs 우→좌, 상→하 vs 하→상)가 설정과 다름
5. **BH1 Function 꺼짐** — 고급 설정의 BH1(브릿지 헤드) 기능 비활성화

## 해결 절차

### 1. 화면 크기 재확인

1. [설정] 탭 → **전광판 화면 구성**
2. `가로` × `세로` 값 = 실제 배치된 모듈 개수 일치 확인
   - 예: 가로 6 × 세로 2 (모듈 당 16×16) = 총 96×32 픽셀
3. [전송] 후 현장 표출 확인

### 2. 전원·케이블 점검

- 꺼진 모듈의 전원 LED 표시등 확인
- 모듈 간 Hub 케이블 연결 상태 재확인 (어느 모듈부터 안 나오는지)
- 첫 번째 꺼진 모듈이 원인 지점

### 3. Scan Order 테스트

1. [설정] 탭 → 표출신호 설정 → `Scan Order` 값 변경
2. 일반적으로 `138 IC` 또는 모듈 스펙에 명시된 값
3. 잘못된 Scan Order면 모듈 내부가 재배열돼 일부만 나옴

### 4. 테스트 패턴

- 숫자 1, 2, 3... 을 모듈 단위로 표출해 순서 확인
- 순서가 뒤섞여 나오면 배선·Scan Order 재조정

### 5. 고급 설정 확인

- [고급] 탭 → **보드기능설정** → `BH1 Function` 활성화 여부
- Off 상태면 하위 체인 모듈이 신호를 못 받을 수 있음
