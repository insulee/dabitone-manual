# 색 반전·RGB 순서 문제

풀컬러 전광판인데 빨간색이 파란색처럼, 노란색이 보라색처럼 나오거나, 특정 색 채널이 완전히 안 보이는 경우.

## 증상

- 빨강 → 파랑으로 표시 (또는 역)
- 흰색이 노랑·청록으로 표시
- 녹색 또는 파랑 채널이 아예 꺼짐
- 모듈별로 색이 다르게 나옴

## 원인

1. **Color Order 설정 불일치** — 모듈 제조사 스펙 (RGB / BGR / GRB 등)과 DabitOne `Color Order`가 어긋남
2. **BPP(색상 깊이) 불일치** — 풀컬러 모듈인데 3BPP·단색 설정으로 전송
3. **모듈 중 일부만 다른 버전** — 공급 배치가 달라 구동 IC가 섞임
4. **RGB LED 개별 고장** — 한 채널만 꺼진 픽셀 다수 존재

## 해결 절차

### 1. Color Order 재확인

1. [설정] 탭 → **표출신호 설정** 영역에서 `Color Order` 드롭다운 확인
2. 기본값 `RGB` → 모듈 스펙이 `BGR`이면 변경 후 [전송]
3. 현장에서 빨강 블록을 띄워 색 확인 → 맞을 때까지 RGB/BGR/GRB/... 순차 테스트

### 2. BPP 재확인

- 모듈이 풀컬러면 **24BPP** 필수. 3BPP(8색)나 1BPP 선택 시 색 손실.
- [설정] 탭 → **전광판 화면 구성** → BPP 드롭다운에서 `24BPP(풀컬러)` 선택.

### 3. 테스트 패턴 전송

- 단순 "RED", "GREEN", "BLUE" 순차 표출로 각 채널이 옳은 위치에 있는지 확인.
- 모든 모듈이 같은 색을 보여야 함. 특정 모듈만 다르면 해당 모듈 펌웨어·Color Order 개별 확인.

### 4. 개별 픽셀 고장

- 여러 이미지에서 같은 좌표가 같은 색만 안 보이면 해당 픽셀 LED 고장. 모듈 교체.
