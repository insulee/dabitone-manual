# 보드 기능 설정

> [!abstract] 컨트롤러 보드 저수준 하드웨어 옵션 조정
> `고급` 탭 > `보드 기능` 섹션. 설치 초기 또는 제조사 지원 요청 시에만 사용.

> [!warning] 전문가용. 잘못 설정하면 컨트롤러 응답 없음·화면 깨짐. 변경 전 현 값 반드시 메모.

### 화면 구성

- `디버깅 모드` — 내부 로그 출력 활성화
- `BH1 모드` — BH1 핀 동작 (입력/출력/특수)
- `J2 (외부 센서)` — 온도/조도/미사용
- `J3 (외부 장치)` — 외부 장치 연동 용도
- `직렬 보드레이트` — Serial 포트 속도 (9600~921600)
- `네트워크 보드레이트` — 이더넷 속도 (10/100 Mbps)
- `모듈 스캔 주파수` — Hz 단위
- `공장 초기화` — 보드 설정만 공장 기본값으로

### 절차

- `고급` 탭 > `보드 기능` 섹션
- 각 항목의 현재 값 확인·메모
- 필요한 항목만 변경
- `적용` → 컨트롤러 재부팅 후 반영

### 항목 설명

- `디버깅 모드` — Serial 모니터로 상세 로그. 제조사 요청 시에만 ON
- `BH1` — 추가 제어 신호 핀
- `J2` — 외부 센서 (밝기 센서로 자동 밝기 조절 등)
- `J3` — 외부 장치 (다른 컨트롤러 동기화 등)
- `모듈 스캔 주파수` — 너무 낮으면 깜빡임, 너무 높으면 밝기 저하. 기본값 권장

### 공장 초기화

> [!info] 보드 설정만 초기화. 저장된 메시지·스케줄은 유지. 단 해당 하드웨어 설정에 의존한 동작은 리셋됨.

> [!warning] 변경 후 컨트롤러 재부팅 필수. 보드레이트 변경 시 일시적으로 연결 끊김 발생.
